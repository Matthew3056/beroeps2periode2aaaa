# Afbeeldingen Map

Plaats hier de afbeeldingen van de ramen-gerechten.

## Vereiste Afbeeldingen

- `tonkotsu.jpg` - Tonkotsu Ramen
- `shoyu.jpg` - Shoyu Ramen
- `miso.jpg` - Miso Ramen
- `spicy.jpg` - Spicy Ramen
- `vegetarian.jpg` - Vegetarian Ramen
- `chicken.jpg` - Chicken Ramen
- `default-ramen.jpg` - Standaard fallback afbeelding

## Aanbevelingen

- Formaat: JPG of PNG
- Grootte: 800x600 pixels of groter
- Aspect ratio: 4:3 of 16:9
- Bestandsgrootte: Maximaal 500KB per afbeelding

De applicatie gebruikt `default-ramen.jpg` als fallback wanneer een afbeelding niet gevonden wordt.
